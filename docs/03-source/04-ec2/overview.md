# AWS EC2 Module
## Overview
This module deploys EC2 instances.

Currently, we support the `work` template. We we will define other templates as
they are needed.

## Templates
| Name   | Description                                  |
| ------ | -------------------------------------------- |
| `work` | A developer focued build with Amazon Linux 2 |
